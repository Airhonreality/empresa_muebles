## 🚀 Phase 2: Information Minimization (Suh's 2nd Axiom)
**Goal**: Reduce the "Cost of Metadata" and achieve "Zero-Magic" Emergence.

### 1. The Inversion of Logic (Inference Engine)
*   [ ] **Auto-Context**: Blocks infer their target context from URL path or parent scope.
*   [ ] **Relational Gravity**: Auto-connect fields ending in `_id` to their respective collections.
*   [ ] **DNA Weight Reduction**: Target 60% less boilerplate in `manifest.json`.

### 2. Fractal Symmetry (Recursive Systems)
*   [ ] **Nested Projections**: Enable collections-within-collections using the same block logic.
*   [ ] **Property Inheritance**: Sub-blocks inherit filters and state from parent containers automatically.

### 3. The "Ghost" Schema (Implicit Admin)
*   [ ] **Auto-Projection**: Generate basic CRUD UI for any context detected in the Vault, even if not defined in DNA.
*   [ ] **Schema Discovery**: Tool to export "Ghost" schemas into formal DNA definitions.

### 4. Satellite Synergy (Dynamic Loading)
*   [ ] **Auto-Registry**: Satellite modules register themselves via a specialized loader (eliminating manual imports in core).
*   [ ] **Bridge Hardening**: Ensure CustomActorBridge is resilient to satellite runtime errors.
