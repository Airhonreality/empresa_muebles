# 🛰️ GUÍA DE IGNICIÓN SOBERANA

¡Bienvenido, Arquitecto! Has encendido el motor de la Semilla Agnóstica. Sigue estos pasos en orden para proyectar tu primera realidad.

## 🏁 Paso 1: Configurar el Satélite (`SYSTEM_CONFIG`)
Antes de crear datos, define quién es el satélite.
1.  En la Forja, busca o crea la entidad en el contexto **`SYSTEM_CONFIG`**.
2.  Define el `site_title`, `home_slug` y `theme`.
3.  Cristaliza los cambios. Esto le dará identidad a tu app.

## 🧬 Paso 2: Definir Leyes (`META_CLASSES`)
Crea tus Arquetipos. ¿Qué vas a gestionar? ¿Facturas? ¿Proyectos? ¿Cápsulas de tiempo?
1.  Crea una entidad en **`META_CLASSES`**.
2.  Define sus campos en el JSON (ej: `name`, `amount`, `date`).
3.  Una vez guardado, la Forja se autoconstruirá para este nuevo tipo de materia.

## 🎭 Paso 3: Proyectar Vistas (`VIEW_PROJECTIONS`)
Haz que el mundo vea tu materia.
1.  Crea una entidad en **`VIEW_PROJECTIONS`**.
2.  Define el `path` (ej: `/dashboard`).
3.  Añade bloques en la zona **OMEGA** para diseñar el layout.
