# Modulo: Render Studio

Herramienta de renderizado fotorrealista de espacios: plano/pieza real + foto del espacio del cliente → imagen del mueble instalado, con precision geometrica. Traducido de notas sobre generacion de imagenes con ComfyUI/RunPod — solo el puente tecnico, no la logica de negocio del proyecto de origen (ver `docs/01_arquitectura_paquete.md`). *(Nota: la capability técnica de renderizado ha sido extraída del repositorio local y se provee vía HTTP delegando al proyecto satélite `estudio_multimedia`)*.
