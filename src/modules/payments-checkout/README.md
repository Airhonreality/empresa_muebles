# Modulo: Pagos

Cobro de ventas de muebles a nivel nacional (Colombia), con el terreno listo para pasarelas internacionales cuando el negocio expanda, bajo un contrato de pagos unico.

## Por que vive en `src/modules/` y no en `storage/`

Mismo razonamiento que `../inbox/README.md` — `storage/` es solo datos y documentacion por contrato del engine, este modulo necesita codigo real (contrato, adapters, UI de checkout) junto a su documentacion de diseno.

## Como retomar el trabajo

Empieza por `INDEX.md` en esta misma carpeta.
